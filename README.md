# Netflix-data-Analysis
Exploratory Data Analysis of Netflix dataset using Python,Numpy ,Pandas and matplotlib.

## Project Overview
This project analyzes Netflix movies and TV shows dataset to find insights
about content type, ratings, countries, and yearly growth.

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- Power BI

## Key Analysis
- Movies vs TV Shows distribution
- Top content producing countries
- Year-wise Netflix content growth
- Ratings analysis.

## Files in Repository
- Netflix_Data_Analysis.ipynb
- netflix_cleaned.csv
